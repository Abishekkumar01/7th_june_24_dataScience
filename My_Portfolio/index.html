Insurance Premium Prediction

Organization: Upflairs

Brief Description: Developed a model to predict insurance premiums using a dataset with features like age, sex, BMI, and smoking status. Key steps included data preprocessing (one-hot encoding, categorical mapping), feature engineering (categorizing BMI), and training multiple models (Linear Regression, Decision Tree Regressor, Random Forest Regressor). Evaluation of models was performed to select the best-performing ones. The project also involved creating visualizations and using libraries such as Flask for the web application, Scikit-learn for machine learning, SQLite for data management, and Pandas and Matplotlib for data analysis and visualization.

Email Classification

Organization: Upflairs

Brief Description: Developed an email classification system using machine learning models to identify emails as spam or ham. The project utilizes a Bag of Words model for feature extraction and Bernoulli Naive Bayes for classification. The web interface, built with Flask, allows users to input email messages and receive a classification result. Additional models like MultinomialNB and GaussianNB were evaluated for performance, with detailed metrics and confusion matrix visualizations included for analysis.
