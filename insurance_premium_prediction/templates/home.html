Insurance Premium Charges Prediction

Project Overview

This project aims to predict insurance premiums based on factors like age, sex, BMI, number of children, smoking status, and region, providing accurate and reliable predictions to help understand and manage insurance costs. By addressing unpredictable premiums, lack of transparency, manual calculations, and limited insight into cost drivers, our machine learning models offer clear, automated, and insightful predictions. This enhances transparency, efficiency, and understanding of how different factors influence premiums.

Customer Benefits

Accurate premium predictions aid in financial planning, health management, informed decision-making, and negotiation. Understanding the impact of factors like BMI and smoking status encourages healthier lifestyle choices. Transparent predictions build trust between customers and insurers, leading to higher satisfaction. Overall, the project empowers customers with knowledge for better financial and health-related decisions.

Project Details

Data and Analysis:

Data Preparation: The dataset includes features such as age, sex, BMI, number of children, smoking status, region, and charges. We perform data preprocessing steps like one-hot encoding and mapping categorical variables to numerical values.

Feature Engineering: We categorize BMI into different health classes (Underweight, Healthy Weight, Overweight, Obese) and analyze the average charges for each class.

Model Training:

  • Linear Regression: For modeling the linear relationship between features and charges.
  • Decision Tree Regressor: To capture non-linear relationships and interactions between features.
  • Random Forest Regressor: To improve prediction accuracy through ensemble learning.

Evaluation: Models are evaluated using training and testing scores to assess their performance. The best-performing models are saved for future use.

Visualization: Various visualizations such as heatmaps, bar charts, and pie charts are used to display the distribution of health categories and model performance.

Outcome: By accurately predicting insurance premiums, the project addresses the problem of understanding and managing insurance costs effectively. It provides valuable insights for both customers and insurers, helping in financial planning and decision-making.

Libraries Used:

  • Flask: For creating the web application and handling server-side logic.
  • Scikit-learn: For building and evaluating machine learning models.
  • SQLite: For managing and storing project data.
  • HTML/CSS: For designing and styling the web interface.
  • Numpy: For numerical operations and handling arrays.
  • Pandas: For data manipulation and analysis.
  • Matplotlib: For creating static, interactive, and animated visualizations.
